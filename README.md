# SmartLogisticsAI-Mule4

## Overview
Enterprise-grade MuleSoft integration project demonstrating event-driven order orchestration with AI-enabled predictive logistics.

## Project Features
- HTTP API to receive new orders
- Data transformation using DataWeave
- Database integration (Postgres)
- Async integration placeholder for AI prediction microservices
- Error handling and logging

- Flow diagram: `docs/architecture-diagram.png`

## Setup Instructions
1. Clone repo
2. Open project in Anypoint Studio
3. Configure HTTP Listener and Postgres connector
4. Run project
